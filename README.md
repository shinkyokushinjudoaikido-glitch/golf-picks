# Golf Picks（トップ + 中古クラブ 公開版）

まずトップページと中古クラブ比較の2ページのみ公開する最小構成。
練習場・ゴルフ場・大会ページは「準備中」表示（リンク無効）。

## ページ
- index.html — トップページ
- clubs.html — 中古クラブ比較

## 公開
このフォルダの中身を GitHub にアップ → Vercel と連携して公開。
ビルド不要の静的サイト。Vercel の Framework Preset は「Other」。

## 後から練習場などを追加するには
1. 該当ページのファイルを追加アップロード
2. config.js の null を実際のファイル名に変更
